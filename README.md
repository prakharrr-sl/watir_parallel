# Steps to do parallel:

## Dependency: 
1. Install Ruby: 
```
brew install ruby
```

2. Install Rake:
```
gem install rake
```

3. Install bundler:
```
gem install bundler
```




## Exporting SAUCEEEEEEE:

1. export USER
```
export USER=`Sauce_User`
```

2. export KEY
```
export KEY=`SAUCE_KEY`
```

3. export HOST
```
export HOST=`you know what to do :)`
```


# RUN TESTS IN SAUCE

`rake test_parallel`
